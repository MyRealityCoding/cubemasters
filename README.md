mineweb-ui
===========

Simple ui page, written in HTML and JQuery to show current server information.

## Getting started

To modify the page, you can change the values in the ```config.json``` file:

```json
{
  "name" : "name-of-your-server",
  "ip" : "server-ip",
  "port" : "mineweb-port",
  "forum" : "link-to-forum",
  "map" : "link-to-map",
  "website" : "link-to-your-website"
}
```

That's it!
